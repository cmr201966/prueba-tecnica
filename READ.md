# README.md (instrucciones en texto)
# 1. clona el repo
# 2. crea una rama: git checkout -b feature/update-config
# 3. modifica algo en settings/dev.py
# 4. commitea: git commit -am "Refactor dev config"
# 5. stash cambios no committeados si quieres probar git stash
# 6. simula conflictos haciendo rebase con otra rama
# 7. usa cherry-pick o revert según el caso
# 8. crea PR hacia dev y luego mergea a main
